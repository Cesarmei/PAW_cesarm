Introdução a HTML

HTML é uma linguagem de marcação para a criação de páginas web que significa Hyper Text Markup Language.

Os elementos HTML são os blocos de construção para a criação de páginas web, representados por marcações que os navegadores web não apresentam mas usam-nas para apresentar o seu conteúdo.
